# 6x6 Sudoku project of the University of Valle
6x6 Sudoku with 2x3 blocks with a graphical user interface, keyboard and mouse events. It features an attractive design for the user and a help system, real-time error checker in the scheme, and clear usage instructions.

## Building the Project

This project uses Maven as its build tool.

### Requirements
- Java JDK 8 or later
- Maven installed and available in your system PATH

### Compile the Project

Run the following command from the project's root directory:

```bash
mvn compile
```
### Generate the Executable JAR
```bash
mvn package
```

### Run JAR file
```bash
mvn clean javafx:run
```
